The dialog element has three content sections:
  1. Title:
    <div dialog-title></div>
    Contains the title of the dialog
  2. Content:
    <div dialog-content></div>
    Contains the content of the dialog
  3. Actions:
    <div dialog-actions></div>
    Contains custom actions of the dialog. (If not given the dialog has a default Ok-button in this section)
