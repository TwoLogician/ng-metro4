## Installation

### Install using Angular CLI:

````
ng add ng-metro4
````

### Import module:

Add NgMetro4Module to your imports:

````
import { NgMetro4Module } from 'ng-metro4';

````

imports: [
  BrowserModule,
  FormsModule,
  ReactiveFormsModule,
  NgMetro4Module,
  ...
]
````

### Generate layout

````
ng g ng-metro4:layout
````
